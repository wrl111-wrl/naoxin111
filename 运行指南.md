# 🎓 教学管理系统运行指南

## 📋 系统概述

这是一个基于Vue.js + Spring Boot的前后端分离教学管理系统，包含管理员、教师、学生三种角色，支持学生信息管理、请假申请等功能。

## 🛠️ 环境要求

### 必须安装的软件：

1. **Java JDK 8+**
   - 下载地址：https://www.oracle.com/java/technologies/downloads/
   - 配置JAVA_HOME环境变量

2. **Node.js 12+**
   - 下载地址：https://nodejs.org/
   - 安装时选择"Add to PATH"

3. **MySQL 5.7+**
   - 下载地址：https://dev.mysql.com/downloads/mysql/
   - 安装后设置root用户密码为 `1234`

4. **Maven 3.6+**
   - 下载地址：https://maven.apache.org/
   - 配置MAVEN_HOME环境变量

## 🗄️ 数据库准备

1. 打开MySQL命令行或MySQL Workbench
2. 执行以下SQL命令：
```sql
CREATE DATABASE DB061 CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

3. 导入数据：
   - 打开 `d:\Project\houduan\database\db061.sql`
   - 在MySQL中执行该SQL文件

## 🚀 快速启动

### 方式一：使用启动脚本（推荐）

双击运行：**`d:\Project\启动项目.bat`**

该脚本会自动：
- 检查环境配置
- 启动后端服务（端口8888）
- 启动前端服务（端口9999）

### 方式二：手动启动

#### 第一步：启动后端服务
```bash
cd d:\Project\houduan
mvn spring-boot:run
```

等待看到以下日志表示启动成功：
```
Started DemoApplication in 5.234 seconds
```

#### 第二步：启动前端服务
打开新的命令行窗口：
```bash
cd d:\Project\houtai
npm install
npm run dev
```

## ❗ 常见问题

### Q1: 后端启动失败
**解决方案：**
- 检查MySQL服务是否启动
- 确认数据库`DB061`已创建
- 检查application.yml中的数据库连接配置

### Q2: 前端启动失败
**解决方案：**
- 删除`node_modules`文件夹后重新执行`npm install`
- 检查Node.js版本是否为12+
- 清理npm缓存：`npm cache clean --force`

### Q3: 登录失败
**解决方案：**
- 确认数据库中已导入测试数据
- 检查用户名密码是否正确
- 查看浏览器控制台错误信息

### Q4: 页面显示空白
**解决方案：**
- 检查后端服务是否正常运行
- 确认前端代理配置正确
- 清除浏览器缓存后重试
